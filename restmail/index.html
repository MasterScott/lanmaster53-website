RESTmail

This is a React driven web interface for the restmail.net API. Restmail.net is a RESTful web service for quickly recieving mail at arbitrary email addresses. To use this interface, have mail sent to any <username>@restmail.net email address. Then, use the username in the form below to view and delete the mail.

CORS is currently not available for the restmail.net API. Therefore, in order to use this inferface, use a proxy (i.e. Burp) to add the following headers to all responses and access this page through the proxy.

Access-Control-Allow-Origin: *
Access-Control-Allow-Methods: GET, DELETE
